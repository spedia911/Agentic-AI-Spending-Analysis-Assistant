## Upload Checklist

- Video is 5 minutes or less.
- Title matches this file or a close variant.
- Description includes the public project link.
- Description mentions the Concierge Agents track.
- Chapters match the final recording timing.
- Cover image and video are attached to the Kaggle Media Gallery.
- YouTube link is pasted into the final Kaggle form checklist.
- `npm run submission:final` passes with the final YouTube URL, project link, video duration, and Kaggle form confirmations.
- Kaggle Writeup is submitted, not left as a draft.
- No unintended private Kaggle Resources are attached to the public Writeup.

## Privacy Checklist

- No `.env` values are visible.
- No service-account JSON is visible.
- No API keys are visible.
- No real account numbers are visible.
- No private financial screenshots are visible.
- No private local machine paths are visible.
- Demo data is sanitized or sensitive values are fully masked.
