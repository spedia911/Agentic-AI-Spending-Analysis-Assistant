# Media Gallery Plan

Kaggle requires a cover image and a video in the Writeup Media Gallery.

## Cover Image

Use [cover-image.svg](cover-image.svg) as the cover art or as the base for a PNG export.

Recommended alt text:

> Agentic AI Spending Analysis Assistant workflow from Drive screenshots through AI extraction, human review, Google Sheets, and dashboard insights.

## Required Video

- Publish the demo video to YouTube.
- Keep it 5 minutes or less.
- Attach it to the Kaggle Writeup Media Gallery.
- Use [VIDEO_SCRIPT.md](VIDEO_SCRIPT.md) as the recording guide.
- Use [YOUTUBE_UPLOAD_METADATA.md](YOUTUBE_UPLOAD_METADATA.md) for the YouTube title, description, chapters, tags, and visibility checklist.

## Optional Screenshots

Use sanitized demo data for every screenshot.

Recommended shots:

1. Dashboard action center with setup health and next-best actions.
2. Technical architecture image from [architecture-diagram.svg](architecture-diagram.svg).
3. Staged import page showing source include/exclude and editable rows.
4. Spending Explorer with category pie chart and transaction rows.
5. Anomaly review panel with related records.
6. Source evidence page with cached preview or Drive handoff.
7. Google Sheets tab list showing durable memory tabs.

## Privacy Checklist

- No `.env` values.
- No service account JSON.
- No API keys.
- No real account numbers.
- No private financial screenshots.
- No local machine paths in public images.
